API Server Running

This is the API server for ckallum.com. The main website is hosted at ckallum.com.

API endpoints:

  • /api - Health check
  • /api/auth-challenge/:pageId - Get authentication challenge
  • /api/verify-password - Verify password
  • /api/messages/:pageId - Get messages for a page
  • /api/messages - Post a new message
